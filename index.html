how to use GIT

To add a file in your GitHub Account you have to do the following steps::


Installing the Git

First you need to install git from the following URL
git desktop


Creating a Repository

In your GitHub account on GITHUB.COM create a new repsitory in its code section you can will see git url copy it

Creating File

Now create a file in VS Code then type the following commands

Git Commands

You have to enter the following commands in terminal of VS Code
git init ////this will initialize git
git add "file_name" /////this will add the file to repository
git clone "url of repository" ////this will clone your current folder with your repository
git commit -m "any message" ////this will commit your file in repsitory
git push ////this will push your file to the online repository

Now your work is online
